1) Rodzaje okienek
alert - wyświetlająca okienko dialogowe z informacją i przyciskiem OK,

prompt – generująca okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.

confirm - służy do tego, aby użytkownik coś potwierdził

2) Konwersja liczb
- parseFloat(zmienna) - konwersja zmiennej do typu float

- parseInt(zmienna) - konwersja zmiennej do typu int (liczbowego)

- toString(zmienna) - konwersja zmiennej do typu łańcuchowego

3) Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:
- Musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez polskich znaków), cyfry oraz "$" i "_".

- Wielkość liter ma znaczenie.

- Należy nie używać słów kluczowych JavaScript jako nazw zmiennych.

4) Typy zmiennych:
- String - Typ tekstowy

- Boolea - Typ logiczny

- Null - Pusta zawartość

- Undefined - Typ niezidentyfikowany
