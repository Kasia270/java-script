1. Określanie właściwości obiektu

Sposób 1 - zapis kropkowy

auto.marka = "Opel";
auto.rok = 1996;

Sposób 2 - zapis tablicowy

auto["cena"] = 25000;

2. Odwołanie do pól i metod

Do pól i metod odwołujemy się przez nazwę obiektu i kropkę.

console.log(samochod1.kolor);
samochod1.jedz();

3. Cechy charakterystyczne konstruktora

  • Wywoływany automatycznie w chwili tworzenia obiektu
  • Nazywa się tak samo jak klasa
  • Nie zwraca wartości
  • Może przyjmować argumenty, jak zwykła funkcja lub metoda

5. Znaczenie słowa this

Słowo this pozwala odwołać się do pól i metod aktualnego obiektu. Pola i metody oddzielamy przecinkami.

6. Konstruktor w JavaScript

function Klient(nazwisko_k, imie_k, zawod_k)
{
 this.nazwisko = nazwisko_k;
 this.imie = imie_k;
 this.zawod = zawod_k;

 wypisz = function()
{
 alert(this.nazwisko + ' ' + this.imie)
}
}

7. Słowo kluczowe new

Słowo new służy do tworzenia nowego obiektu na podstawie konstruktora.

var osoba1 = new Klient("Kowalski", "Jan", "kierowca");

8. Prototype

Prototype umożliwia dodawanie metod i właściwości wspólnych dla wszystkich obiektów danej funkcji.

 function klient()
{
 this.nazwisko='Bielski';  this.name='Paweł';
}
klient.prototype.pisz_dane = function()
{
 document.write(this.nazwisko + ' ' + this.imie);
}
 klient.prototype.zawod = 'kierowca';
 var osoba1 = new klient();
 osoba1.pisz_dane();
